¿Qué son los códigos de estado de respuestas HTTP? Realizar una tabla con su clasificación. Bibliografía MDN

Respuestas Informativas
100 Continue Indica que todo esta bien
101 Switching Protocol Este código se envía en respuesta a un encabezado de solicitud
102 Processing Este código indica que el servidor ha recibido la solicitud
103 Early Hints Codigo para ser usado en un encabezado
Respuestas Sastifactorias
200 Ok La solicitud ha tenido éxito
201 Created La solicitud ha tenido éxito
202 Accepted La solicitud se ha recibido
203 Non-Authoritative Information La petición se ha completado con éxito
204 No content La petición se ha completado con éxito pero su respuesta no tiene ningún contenido
205 Reset content La petición se ha completado con éxito, pero su respuesta no tiene contenidos
206 Partial Content La petición servirá parcialmente el contenido solicitado
207 Multi-Status Una respuesta Multi-Estado
208 Multi-Status El listado de elementos DAV ya se notificó previamente
226 Im used El servidor ha cumplido una petición
Redireccion
300 Multiple Choice Esta solicitud tiene más de una posible respuesta
301 Moved Permanently Este código de respuesta significa que la URI del recurso solicitado ha sido cambiado.
302 Found Este código de respuesta significa que el recurso de la URI solicitada ha sido cambiado temporalmente
303 See other El servidor envía esta respuesta para dirigir al cliente a un nuevo recurso
304 Not Modified Esta es usada para propósitos de "caché"
305 Use proxy Fue definida en una versión previa de la especificación del protocolo HTTP
306 Unused Este código de respuesta ya no es usado más.
307 Temporary Redirect El servidor envía esta respuesta para dirigir al cliente a obtener el recurso solicitado a otra URI
308 Permanent Redirect Significa que el recurso ahora se encuentra permanentemente en otra URI
Errores del cliente
400 Bad Request Esta respuesta significa que el servidor no pudo interpretar la solicitud
401 Unauthorized Es necesario autenticar para obtener la respuesta solicitada
402 Payment Required Este código de respuesta está reservado para futuros usos
403 Forbidden El cliente no posee los permisos necesarios para cierto contenido
404 Not Found El servidor no pudo encontrar el contenido solicitado
405 Method Not Allowed El método solicitado es conocido por el servidor pero ha sido deshabilitado
406 Not Acceptable Esta respuesta es enviada cuando el servidor, aplica Una negociación de contenido servidor-impulsado no encuentra ningún contenido
407 Proxy Authentication Required Esto es similar al código 401
408 Request Timeout Esta respuesta es enviada en una conexión inactiva en algunos servidores
409 Conflict Esta respuesta puede ser enviada cuando una petición tiene conflicto con el estado actual del servidor.
410 Gone Esta respuesta puede ser enviada cuando el contenido solicitado ha sido borrado del servidor.
411 Length Required El servidor rechaza la petición ya que el encabezado no esta definido.
412 Precondition Failed El cliente ha indicado pre-condiciones en sus encabezados la cual el servidor no cumple.
413 Payload Too Large La entidad de petición es más larga que los límites definidos por el servidor
414 URI Too Long La URI solicitada por el cliente es más larga de lo que el servidor está dispuesto a interpretar.
415 Unsupported Media Type El formato multimedia de los datos solicitados no está soportado por el servidor
416 Requested Range Not Satisfiable El rango especificado por el campo de encabezado Range en la solicitud no cumple
417 Expectation Failed Significa que la expectativa indicada por el campo de encabezado Expect solicitada no puede ser cumplida por el servidor.
418 I'm a teapot El servidor se rehúsa a intentar hacer café con una tetera.
421 Misdirected Request La petición fue dirigida a un servidor que no es capaz de producir una respuesta.
422 Unprocessable Entity La petición estaba bien formada pero no se pudo seguir debido a errores de semántica
423 Locked El recurso que está siendo accedido está bloqueado
424 Failed Dependency La petición falló debido a una falla de una petición previa
426 Upgrade Required El servidor se rehusa aplicar a la solicitud usando el protocolo actual, a no ser que el cliente use otro protocolo
428 Precondition Required El servidor origen requiere que la solicitud sea condicional
429 Too Many Requests El usuario ha enviado demasiadas solicitudes en un periodo de tiempo dado.
431 Request Header Fields Too Large El servidor no está dispuesto a procesar la solicitud porque los campos de encabezado son demasiado largos
451 Unavailable For Legal Reasons El usuario solicita un recurso ilegal, como alguna página web censurada por algún gobierno
Errores de servidores
500 Internal Server Error El servidor ha encontrado una situación que no sabe cómo manejarla.
501 Not Implemented El método solicitado no está soportado por el servidor y no puede ser manejado
502 Bad Gateway Esta respuesta de error significa que el servidor, obtuvo una respuesta invalida
503 Service Unavailable El servidor no está listo para manejar la petición
504 Gateway Timeout Esta respuesta de error es dada cuando el servidor está actuando como una puerta de enlace y no puede obtener una respuesta a tiempo.
505 HTTP Version Not Supported La versión de HTTP usada en la petición no está soportada por el servidor.
506 Variant Also Negotiates negociación de contenido transparente para la petición resulta en una referencia circular.
507 Insufficient Storage negociación de contenido transparente para la petición resulta en una referencia circular.
508 Loop Detected El servidor detectó un ciclo infinito mientras procesaba la solicitud.
510 Not Extended Extensiones adicionales para la solicitud son requeridas para que el servidor las cumpla.
511 Network Authentication Required El código de estado 511 indica que el cliente necesita autenticar para obtener acceso a la red.
